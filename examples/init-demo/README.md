# Init Demo — Falsifiable Happy Path

This example proves the README claims. Run it locally and verify every step.

## What the README Promises

> `evalgate init` detects your Node project, runs your tests to create a baseline, installs a GitHub Actions workflow, and prints what to commit. Open a PR and CI blocks regressions automatically.

Here’s exactly what you get.

| File | Purpose |
|------|---------|

#### `.github/workflows/evalgate-gate.yml`

```yaml
# EvalGate Regression Gate
# Auto-generated by: npx evalgate init
name: EvalGate Gate

on:
  pull_request:
    branches: [main]

concurrency:
  group: evalgate-${{ github.ref }}
  cancel-in-progress: true

jobs:
  regression-gate:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-node@v4
        with:
          node-version: '20'
          cache: npm
      - run: npm ci
      - name: EvalGate Doctor (preflight)
        continue-on-error: true
        run: npx -y @evalgate/sdk@3.2.2 doctor
      - name: EvalGate Regression Gate
        run: npx -y @evalgate/sdk@3.2.2 gate --format github
      - name: Upload report
        if: always()
        uses: actions/upload-artifact@v4
        with:
          name: evalgate-report
          path: |
            evals/regression-report.json
            .evalgate/last-report.json
          if-no-files-found: ignore
```

## How to Verify (Falsify)

1. **Clone and init:**
   ```bash
   cd examples/init-demo
   npm install
   npx evalgate init
   ```

2. **Confirm files exist:**
   ```bash
   ls evals/baseline.json evalgate.config.json .github/workflows/evalgate-gate.yml
   ```

3. **Run gate locally:**
   ```bash
   npx evalgate gate --format github
   ```
   Expect: `✅ EvalGate Gate: pass`

4. **Break a test** (e.g. change `assert.strictEqual(1, 1)` to `assert.strictEqual(1, 2)` in `test.js`):
   ```bash
   npx evalgate gate
   ```
   Expect: `❌ EvalGate Gate: regression` with exit code 1

5. **Restore and push** — open a PR; CI runs the same gate and blocks merge if tests fail.
